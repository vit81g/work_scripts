    Импорт библиотек:

        subprocess - для выполнения системных команд.

        os - для работы с файлами.

    Функция execute_commands(filename):

        Открывает файл filename для чтения.

        Считывает все строки файла в список commands.

        Создает файл command_output.txt для записи результатов.

        Итерируется по каждой команде в списке:

            Удаляет лишние пробелы с помощью command.strip().

            Проверяет, не пуста ли команда (if command:).

            Выполняет команду с помощью subprocess.check_output():

                shell=True - позволяет интерпретировать команду как строку оболочки.

                stderr=subprocess.STDOUT - объединяет стандартный вывод и ошибки в один поток.

            Если команда выполнена успешно, записывает команду и результат в output_file.

            Если произошла ошибка, записывает команду и сообщение об ошибке в output_file.

    Пример использования:

        Замените filename на имя вашего файла с командами.

        Вызовите функцию execute_commands(filename).

Как использовать:

    Создайте файл commands.txt:

        В каждой строке файла запишите одну команду Unix.

        Например:

              
        ls -l
        whoami
        date

            

    Use code with caution.

Запустите скрипт Python:

    Сохраните код в файл .py (например, command_executor.py).

    Запустите скрипт в терминале: python command_executor.py

Просмотрите результат:

    После выполнения скрипта откройте файл command_output.txt, чтобы просмотреть результаты выполнения команд.
