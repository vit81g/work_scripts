**************************** Код возврата ******************
True это 0
False это все что отлично от 0

Переменные в bash начинаются с символа $
Переменная, которая содержит код возврата последней выполненой команды, это символ ?

Пример выполнения команды ls -l (успешно отработала):
echo $?
# Выведет код возврата 0
При неуспешном срабатывании выведет отличное от 0 число:
# Код возврата равен 2


********************** Логическое И AND ********************
&& логическое И AND

Пример:
# При успешной отработки первой команды ls -l, выполнится вторая команда echo OK
└─$ ls -l && echo OK

Результат:
-rw-r--r-- 1 vit81 vit81  4231 Mar 25 16:52 work_file.txt
OK

Пример:
# При ошибке вывода первой команды ls -l asafg, вторая команда не будет выполнена
└─$ ls -l asafg && echo OK

Результат:
ls: cannot access 'asafg': No such file or directory

******************** Логическое ИЛИ OR ********************
|| логическое ИЛИ OR
Если не выполнится левая часть команды, то тогда выполнится правая часть команды

Пример:
# левая часть команды ls -l выполнена без ошибок
└─$ ls -l || echo OK     

Результат:
-rw-r--r-- 1 vit81 vit81  4231 Mar 25 16:52 work_file.txt

Пример:
# левая команда ls -l asafg выполнена с ошибкой
└─$ ls -l asafg || echo ERR

Результат:
ls: cannot access 'asafg': No such file or directory
ERR

******************** Логическое НЕ ИМЕЕТ ЗНАЧЕНИЕ ***********
; логическое НЕ ИМЕЕТ ЗНАЧЕНИЕ
Выполнена будет в любом случае, независимо от того будет ли выполнена первая команда без ошибок или вторая и т.д.

Пример:
─$ ls -l ; echo OK 

Результат:
-rw-r--r-- 1 vit81 vit81  4231 Mar 25 16:52 work_file.txt
OK

Логические операторы работают на основании кода возврата программы (0 - True; не 0 - False)
&& - логическое "И"
|| - логическое "ИЛИ"
;  - логическое "НЕ ИМЕЕТ ЗНАЧЕНИЯ"

Итоги:
1. Команда отработала без ошибок если
Код возврата равен 0

2. Как выполнить команду с условием что предыдущая выполнилась без ошибок?
com1 && com2

3. Как выполнить команду с условием что предыдущая выполнить с ошибкой?
com1 || com2

4. Объясните условия конструкции (com1 || com2) && com3
com3 выполнится только тогда когда успешно выполнится либо com1, либо com2

echo $?
