
Для контроля статуса всех сервисов в ОС RHEL7 следует использовать команду 

systemctl list-unit-files
----------------------------------------------------------------------

Проверить отключение ipv6 следует командой: 

ip addr|grep inet6
----------------------------------------------------------------------

Проверка отсутствия в ОС пакетов telnet-server

rpm -qa | grep telnet

Должны отсутствовать пакеты типа  telnet-server-0.17-48.el6.x86_64
-----------------------------------------------------------------------

Проверка отсутствия пакетов серверов ftp и его аналогов

rpm -qa | grep ftp

Должны отсутствовать пакеты типа ftp-server-0.49-8.el6.x86_64
------------------------------------------------------------------------

Для проверки статуса компонеты ОС SELinux необходимо выполнить команду

sestatus
-------------------------------------------------------------------------

Контроль настроек sudo

sudo more /etc/sudoers
-----------------------------------------------------------------------------

Для проверки статуса всех УЗ системы необходимо выполнить команду

for users in $(cut -f1 -d: /etc/passwd);do  sudo passwd -S "$users"; done
----------------------------------------------------------------------------------
